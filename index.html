Tweet Wall FX

Crowd Entertainment for your Conference

Big Screen Mode

TweetWallFX scales to full blown theater rooms - select what you like to display on your screen from all views available.

Entertaining

Keep your information displays interesting with the innovative step engine concept for sequencing animations

Configurable

No need to write any code - all views and animations of TweetWallFX can be fully configured using a JSON configuration format.

Cross Platform

Only a small budget - TweetWallFx can be run on as small hardware as avaiable, e.g. RaspberryPi using OpenJDK and JavaFX.

Tweet Wall FX Archeology

A shot trip through time and space

november 2014

First public Tweet Wall FX

After some challenging problems the first version of the Tweet Wall FX was shown on the exhibition floor of Devoxx Belgium 2014. The shown version was based on the work of a world wide community  led by Johan Vos, Jos√© Pereda, Jason Polastrini and Sven Reimers and finished in the Hackergarten on site at Devoxx, sponsored by JCP.

may 2015

Tweet Wall FX organization created at GitHub by JUG Bodensee

Serving the request for another version of the wall for the 2015 edition of Devoxx, JUG Bodensee as a community took over the development effort.

october 2015

1st Tweet Wall FX at JavaOne Hackergarten

With the wall being an early adopter for all types of technologies on a JVM and Java language level it was the perfact candidate for the JavaOne Hackergarten. Running on a RaspberryPi people were invited to have look at the technologies used and create new features to be integrated into the wall.

october 2016

A new Era: Initial StepEngine released

Creating sequences of different animations became way to complex to be maintained easily. As a solution a StepEngine was created as the core of all sequencing. This allowed a centralized setup of the whole animation sequence driving the wall display.

november 2016

Image Mosaic in production

Up to now the wall was only about displaying tweets - with the Image Mosaic the wall follows the trend of pushing images to tell a story.

march 2017

The Wall is now International

After serving Devoxx Belgium for some time, a first request from another conference was served by the community - Tweet Wall FX for DevoxxUS

november 2017

Prime Time for the Big Screen

The first time on the big scrreen - Devoxx 2017 features the Wall on the cinema screens - including a freshly developed client API to retrieve information from the Devoxx CFP API.

october 2018

The Next Generation

After a major restructuring effort all core functionalities of the Wall are now available for reuse using JCenter. This allows for building new Walls by just creating a new configuration.

november 2018

Devoxx Keynote

After years of hard work the integration of AI technologies to prevent some hashtag hijacking as happened in 2018 brught the Wall onto the keynote stage!
